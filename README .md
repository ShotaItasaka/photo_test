# filgraph

## サイト概要
### サイトテーマ
  フィルム写真の共有を主な目的とし、使用したカメラごとにジャンル検索をできるようにする。
​
### テーマを選んだ理由
  フィルムカメラを始めてみたいけど、どんな機種を選べばいいかわからない。この疑問をフィルム写真の投稿に特化したサイトを作成することで<br>
​ 参考にしやすく、選択に困りづらくなるとおもったから。

### ターゲットユーザ
  フィルムカメラをこれから始めたいと思っている方
  今、フィルムカメラをやってる方
​
### 主な利用シーン
  撮影した作例を共有する。
  購入を迷っている方は共有されている写真で撮れる写真のイメージを掴む。
